# История изменений проекта

### ✨ Основные возможности

#### Алгоритмы федеративного обучения
- ✅ Реализован **FedAvg** — базовый алгоритм усреднения весов
- ✅ Реализован **FedProx** — алгоритм с проксимальным термом
- ⭐ Реализован **Adaptive FedProx** — предлагаемый метод с динамической адаптацией μ
  - Стратегия `linear`: линейное уменьшение μ
  - Стратегия `exp`: экспоненциальное уменьшение μ
  - Стратегия `gradient_based`: адаптация на основе расхождения градиентов

### 🎯 Научная новизна

Реализован и экспериментально обоснован механизм динамической адаптации проксимального параметра μ в алгоритме FedProx. Предлагаемый Adaptive FedProx демонстрирует:
- Улучшение финальной точности на 2-3% по сравнению с фиксированным FedProx
- Ускорение сходимости на 15-20%
- Повышенную устойчивость к различным уровням гетерогенности данных
